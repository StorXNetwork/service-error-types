Storj Service Error Types
=========================

Common error types for various Storj services.

```
npm install storj-service-error-types --save
```

```js
var errors = require('storj-service-error-types');
```

Error Types
-----------

* `NotFoundError`
* `NotAuthorizedError`
* `NotImplementedError`
* `BadRequestError`
* `InternalError`
